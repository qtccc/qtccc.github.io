Predicting Customer Churn Using K-Nearest Neighbors

In this project, I developed a predictive model to identify customers at risk of churning (leaving the bank) using the K-Nearest Neighbors (KNN) algorithm. The dataset includes customer demographics, account balances, and salary information. I performed data preprocessing, including handling missing values and feature scaling, followed by feature selection to improve model accuracy. The KNN model was implemented from scratch, with hyperparameter tuning and cross-validation applied to optimize the choice of K and distance metrics (e.g., Euclidean, Manhattan). The model's performance was evaluated using metrics such as accuracy, precision, recall, F1-score, and the ROC curve.
